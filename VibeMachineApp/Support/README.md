# VibeMachine サポート

VibeMachineは、写真を端末の外に送らないAI写真加工アプリです。

## よくある質問

**Q. 写真はどこかにアップロードされますか？**
いいえ。シーン認識・美容加工・AI顔補正・フィルターを含むすべての処理は端末内で完結し、写真や顔データが外部に送信されることはありません。

**Q. 広告を消すには？**
加工画面右下の「広告を削除」から買い切りで購入できます。機種変更後や再インストール後は、同じ画面の「購入を復元」で復元できます。

**Q. 無料で使える機能に制限はありますか？**
ありません。すべての加工機能を無料でお使いいただけます。
