# VibeMachine プライバシーポリシー

最終更新日: 2026年7月3日

VibeMachine（以下「本アプリ」）は、プライバシーを最優先に設計された写真加工アプリです。
本ポリシーは、本アプリが取り扱う情報とその利用目的を説明します。

## 写真について — 端末の外に送信されません

本アプリの画像処理（シーン認識、美肌・デカ目・小顔などの美容加工、AI顔補正、フィルター）は、
**すべてお使いの端末内で完結します**。

- 写真・画像データを外部サーバーにアップロードすることは一切ありません
- 顔認識・顔補正の処理結果や顔データが端末外に送信されることも一切ありません
- 加工した写真は、お客様の操作によってのみカメラロールに保存されます

## 収集する情報

本アプリは、サービスの改善と広告の表示のために以下の情報を利用します。
いずれも**個人を特定する情報や写真の内容は含みません**。

### 1. 利用状況の分析（Google Analytics for Firebase）

アプリの改善のため、操作イベント（例: 写真を読み込んだ、フィルターを適用した、
保存した）とその属性（シーンの種類、フィルター名、加工強度の数値など）、
および端末の機種・OSバージョン等の技術情報を収集します。
広告識別子（IDFA）は使用しません。

### 2. クラッシュ情報（Firebase Crashlytics）

アプリの不具合修正のため、クラッシュ発生時の技術的情報（スタックトレース、
端末機種、OSバージョン等）を収集します。

### 3. 広告（Google AdMob）

本アプリは広告収入により無料で提供されています。広告の表示に伴い、
Google AdMobがIPアドレス等の技術情報を受け取る場合があります。
本アプリは**パーソナライズ広告を使用せず**、App Tracking Transparencyによる
トラッキング許可も求めません（トラッキングを行わないため）。

広告除去（買い切り）をご購入いただくと、広告は表示されなくなります。
