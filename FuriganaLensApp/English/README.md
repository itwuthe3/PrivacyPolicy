**Privacy Policy**

This privacy policy applies to the "FuriganaVision" application (hereinafter "the App") created by Takahiro Ito (hereinafter "Service Provider"). This service is intended to be used "as is" without modification.

**Information Collection and Use**

The App collects the following information:

* Image data captured from the camera (for character recognition processing)
* Application usage statistics (access times, usage duration, number of recognitions performed, etc.)
* Basic device information (OS version, device type, etc.)
* App crash reports and error logs (for improving app stability)

This data is used for the following purposes:

* Character Recognition Function: Recognizes kanji characters from camera-captured images and generates furigana. Image data is processed only on the device and is never transmitted externally
* Service Improvement: Application usage statistics are utilized for feature improvements and bug fixes
* Notifications: Important information and update notifications may be sent

**Important: Camera Data Handling**

* Images captured by the camera are used only for real-time character recognition processing
* Captured images are processed on the device and are never saved
* Image data is never transmitted to external servers
* Recognized text data is also processed only on the device and never transmitted externally
* The App operates offline and does not require an internet connection

This App is designed with privacy protection as the highest priority and guarantees the following:
