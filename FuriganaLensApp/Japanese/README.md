**プライバシーポリシー**

このプライバシーポリシーは、Takahiro Ito（以下「サービス提供者」）が作成した「ふりがなビジョン」アプリケーション（以下「本アプリ」）に適用されます。このサービスは「現状のまま改変せず」使用することを意図しています。

**情報の収集と使用**

本アプリは、以下の情報を収集します：

* カメラから取得した画像データ（文字認識処理用）
* アプリケーションの使用状況（アクセス日時、使用時間、認識実行回数など）
* デバイスの基本情報（OSバージョン、デバイスタイプなど）
* アプリのクラッシュレポートやエラーログ（アプリの安定性向上用）

これらのデータは以下の目的で使用されます：

* 文字認識機能：カメラで撮影した画像から漢字を認識し、ふりがなを生成します。画像データはデバイス内でのみ処理され、外部に送信されることはありません
* サービスの改善：アプリケーションの使用状況データは、機能改善とバグ修正に活用されます
* 通知の送信：重要な情報やアップデートに関する通知をお送りする場合があります

**重要：カメラデータの取り扱いについて**

* カメラで撮影された画像は、リアルタイムで文字認識処理にのみ使用されます
* 撮影された画像はデバイス内で処理され、保存されることはありません
* 画像データが外部サーバーに送信されることは一切ありません
* 認識された文字データもデバイス内でのみ処理され、外部に送信されません
* 本アプリはオフラインで動作し、インターネット接続は必須ではありません

**第三者のアクセス**

本アプリは、プライバシー保護を最優先に設計されており、以下の点を保証します：

* カメラで撮影された画像データは第三者と共有されることはありません
* 認識された文字データも第三者と共有されることはありません
* 集計された匿名の使用統計データのみが、アプリケーションとサービスの改善を支援するために、定期的に外部サービスに送信される場合があります

本アプリケーションは、データの取り扱いに関する独自のプライバシーポリシーを持つ第三者サービスを利用する可能性があります。以下は、本アプリが使用する第三者サービスプロバイダーのプライバシーポリシーへのリンクです：

* [Google プライバシーポリシー](https://www.google.com/policies/privacy/)（Firebase Analytics使用）

サービス提供者は、ユーザーが提供した情報および自動的に収集された情報を以下の場合に開示することがあります：

* 法律で要求される場合（召喚状や類似の法的手続きに従うなど）
* 開示が自社の権利を保護し、お客様や他者の安全を守り、不正行為を調査し、または政府の要請に応じるために必要であると誠実に信じる場合
* サービス提供者に代わって働き、開示された情報を独立して使用せず、このプライバシーステートメントに定められた規則を遵守することに同意した信頼できるサービスプロバイダーとの間で

**カメラ権限について**

本アプリは、書籍や文書の漢字を認識してふりがなを表示するために、カメラ機能を使用します：

* カメラ権限は文字認識機能の提供にのみ使用されます
* ユーザーの明示的な許可なしにカメラにアクセスすることはありません
* カメラで撮影された画像は保存されず、処理後は即座に破棄されます
* 顔認識や個人識別のためにカメラを使用することはありません

**オプトアウトの権利**

本アプリによる情報収集を停止するには、アプリケーションをアンインストールするだけで簡単に行えます。モバイルデバイスやモバイルアプリケーションマーケットプレイス、ネットワークの一部として利用可能な標準的なアンインストールプロセスを使用できます。

**データ保持ポリシー**

* カメラで撮影された画像：リアルタイム処理のみで、保存されません
* 認識された文字データ：アプリ使用中のみメモリに保持され、アプリ終了時に削除されます
* 使用統計データ：お客様がアプリケーションを使用している間、および使用後の合理的な期間、ユーザーが提供したデータを保持します

アプリケーションを通じて提供したユーザーデータの削除を希望される場合は、itwuthe3@icloud.comまでご連絡ください。合理的な時間内に対応いたします。

**セキュリティ**

本アプリは、お客様の個人情報を保護するために、以下のセキュリティ対策を実装しています：

* すべての画像処理はデバイス内で実行されます
* ネットワーク通信は最小限に抑制されています
* データの暗号化と安全な通信プロトコルを使用しています

**変更**

このプライバシーポリシーは、理由の如何を問わず随時更新される場合があります。サービス提供者は、新しいプライバシーポリシーでこのページを更新することにより、プライバシーポリシーの変更をお知らせします。変更を定期的に確認することをお勧めします。継続的な使用は、すべての変更の承認とみなされます。
