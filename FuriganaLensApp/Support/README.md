# ふりがなビジョン　サポートページ

## アプリ概要

このアプリは書籍や文書の漢字を認識して、自動的にふりがなを表示するカメラアプリです。

### 機能
* カメラを使った漢字のリアルタイム認識
* 高精度な日本語形態素解析によるふりがな生成
* 複数の漢字を同時に認識・表示
* 日本語・英語の多言語対応
* シンプルで直感的なユーザーインターフェース
* 一時停止・再開機能
* 認識結果の座標順ソート（文書の読み順に表示）

### 主な用途
* 書籍や雑誌の漢字読み取り支援
* 学習資料の読解サポート
* 日本語学習者の読書補助
* 難読漢字の読み方確認

## 使用方法

### 基本操作
1. アプリを起動すると、カメラ権限の許可を求められます。「許可」を選択してください。
2. カメラが起動するまで数秒お待ちください（初回起動時は少し時間がかかります）。
3. 漢字が含まれた書籍や文書にカメラを向けます。
4. 漢字が認識されると、画面下部に自動的にふりがなが表示されます。
5. 「ていし」ボタンで認識を一時停止、「さいかい」ボタンで再開できます。

### 認識のコツ
* 文字がはっきり見えるよう、適度な距離を保ってください
* 照明が十分な場所で使用してください
* 手ブレを避けるため、カメラを安定して保持してください
* 複雑な背景は避け、文字が明確に見える状態にしてください

### トラブルシューティング

#### カメラが起動しない場合
1. 設定アプリを開く
2. 「ふりがなビジョン」を選択
3. 「カメラ」をオンにする
4. アプリを再起動する

#### 漢字が認識されない場合
* 文字のサイズが適切か確認してください（小さすぎる・大きすぎる文字は認識されにくい場合があります）
* 文字が鮮明に写っているか確認してください
* 照明条件を改善してください
* カメラレンズが汚れていないか確認してください

#### アプリが重い・遅い場合
* 他のアプリを終了して、メモリを確保してください
* デバイスを再起動してください
* アプリを最新版にアップデートしてください

## 対応環境

* iOS 12.0以降
* iPhone・iPad対応
* カメラ機能が必要

## プライバシーについて

本アプリは以下の点でプライバシーを保護しています：

* カメラで撮影した画像は端末内でのみ処理され、外部に送信されません
* 認識された文字データも端末内でのみ処理されます
* ユーザーの個人情報を収集・保存することはありません
* インターネット接続は不要で、オフラインで動作します

詳細は[プライバシーポリシー](../Japanese/README.md)をご確認ください。

## よくある質問

### Q: オフラインで使用できますか？
A: はい、インターネット接続は不要です。すべての処理は端末内で行われます。

### Q: 手書きの文字も認識できますか？
A: 印刷された文字の認識に最適化されています。手書き文字の認識精度は印刷文字より劣る場合があります。

### Q: 縦書きの文書も対応していますか？
A: はい、縦書き・横書きの両方に対応しています。

### Q: 古い漢字体や異体字も認識できますか？
A: 一般的な漢字の認識に最適化されています。古い字体や特殊な異体字は認識されない場合があります。

### Q: 認識精度を向上させる方法はありますか？
A: 以下の点にご注意ください：
- 十分な照明を確保する
- カメラを安定して保持する
- 文字が画面内に収まるよう適切な距離を保つ
- 背景がシンプルな状態で撮影する
