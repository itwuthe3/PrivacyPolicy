# MosaicMachine プライバシーポリシー

最終更新日: 2026年7月6日

MosaicMachine（以下「本アプリ」）は、プライバシーを最優先に設計された動画モザイクアプリです。
本ポリシーは、本アプリが取り扱う情報とその利用目的を説明します。

## 動画について — 端末の外に送信されません

本アプリの動画処理（顔検出、追跡、モザイク・ぼかし・黒塗りの適用、書き出し）は、
**すべてお使いの端末内で完結します**。

- 動画・映像データを外部サーバーにアップロードすることは一切ありません
- 顔検出・追跡の処理結果や顔データが端末外に送信されることも一切ありません
- 加工した動画は、お客様の操作によってのみカメラロールに保存されます

## 顔データの取り扱い

本アプリはモザイクを適用するために端末内で顔検出を行いますが、顔データを収集しません。

- **処理する情報**: AppleのVisionフレームワークによる顔検出で得られる、顔の位置を示す矩形座標のみです。
  顔写真・顔の特徴量（顔認識用データ）は取得も生成もせず、個人を識別する処理は行いません
- **利用目的**: 検出した位置にモザイク・ぼかし・黒塗りを適用するためだけに使用します
- **保存場所**: 顔の位置情報は編集中の一時的なメモリ上にのみ存在します。端末内のストレージにも、
  外部サーバーにも保存されません
- **保持期間**: 編集セッションの間のみです。動画を切り替えるかアプリを終了した時点で破棄されます
- **第三者への共有**: 一切ありません。顔の位置情報・映像・顔データが端末の外に送信されることはありません
- **分析との関係**: 利用状況の分析（下記）に含まれるのは「検出された顔の個数」という数値のみで、
  顔の画像・位置・特徴量が送信されることはありません

## 収集する情報

本アプリは、サービスの改善と広告の表示のために以下の情報を利用します。
いずれも**個人を特定する情報や動画の内容は含みません**。

### 1. 利用状況の分析（Google Analytics for Firebase）

アプリの改善のため、操作イベント（例: 動画を読み込んだ、モザイクを適用した、
保存した）とその属性（動画の長さの秒数、検出された顔の数、スタイルの種類など）、
および端末の機種・OSバージョン等の技術情報を収集します。
広告識別子（IDFA）は使用しません。

### 2. クラッシュ情報（Firebase Crashlytics）

アプリの不具合修正のため、クラッシュ発生時の技術的情報（スタックトレース、
端末機種、OSバージョン等）を収集します。

### 3. 広告（Google AdMob）

本アプリは広告収入により無料で提供されています。広告の表示に伴い、
Google AdMobがIPアドレス等の技術情報を受け取る場合があります。
本アプリは**パーソナライズ広告を使用せず**、App Tracking Transparencyによる
トラッキング許可も求めません（トラッキングを行わないため）。

広告除去（買い切り）をご購入いただくと、広告は表示されなくなります。
