# ネオチン｜添い寝アラーム　サポートページ

## アプリ概要

このアプリはWatch対応のシンプルな添い寝アラームアプリです。

## 説明

主な目的としては、子供と添い寝をする両親が寝落ちしないようにするものです。
Watchでユーザーの睡眠を検知したら、Apple Watchを振動させてユーザーを起こします。

## 仕様

1. スマートフォンアプリとWatchアプリの連携
2. Watchの加速度センサーによる睡眠検知
3. 振動によるサイレントアラーム
4. 簡単な操作による添い寝モードの開始と終了
5. 睡眠データの記録と表示

### 推奨設定
* 動き閾値: 1.00
* 判定時間: 60秒
* 睡眠判定連続回数: 5

## 使用方法

1. アプリを起動し、睡眠検知条件を設定します。
2. メイン画面で「添い寝モード」をONして添い寝モードを開始します。
3. 設定条件に従って睡眠を検知すると、Watchが振動します。
4. メイン画面で「添い寝モード」をOFFすると添い寝モードが終了します。
